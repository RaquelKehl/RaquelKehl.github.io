Arcade

Pipeline — route the data.

A scrambled network, one core, many endpoints. Rotate the tiles until every endpoint is connected — in as few moves as possible. It’s the day job, with better feedback loops.

level 1/5 moves 0 par 0 best —

click / tap a tile to rotate · arrow keys + enter for keyboard play · copper = connected to the core

How to play
  • Goal: connect every tile to the copper core
  • Rotate: click / tap a tile, or arrows + enter
  • Score: fewest moves — par is the scramble’s minimum
  • Best scores are saved on this device only
Why this game
  • Every transformation is this puzzle: the pieces exist, the connections don’t
  • Fewer moves = fewer manual steps — the 60% on the homepage
  • Built in vanilla Canvas 2D, ~0 dependencies, like the rest of the site
